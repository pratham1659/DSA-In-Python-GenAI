## 📖 About

This repository is a self-paced study journal for mastering **Python programming** and **Data Structures & Algorithms (DSA)**. Each topic is covered in its own interactive Jupyter Notebook, with code examples, explanations, and practice problems.

Whether you're a beginner starting with Python or someone brushing up on DSA concepts, this repo has something for you.

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Python 3.x](https://www.python.org/downloads/)
- [Jupyter Notebook](https://jupyter.org/install) or [JupyterLab](https://jupyterlab.readthedocs.io/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/pratham1659/DSA-IN-PYTHON.git
   cd DSA-IN-PYTHON
   ```

2. **Create a virtual environment** (recommended)
   ```bash
   python -m venv venv
   source venv/bin/activate       # On macOS/Linux
   venv\Scripts\activate          # On Windows
   ```

3. **Install dependencies**
   ```bash
   pip install numpy pandas matplotlib seaborn jupyter
   ```

4. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

5. Open any `.ipynb` file and start learning! 🎉
